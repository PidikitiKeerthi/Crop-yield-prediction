Crop Yield Prediction

Overview : This project predicts US corn yield using machine learning models based on weather, soil, and environmental data.

Models Used :
Linear Regression
Decision Tree
Random Forest
SVM
CNN, RNN, LSTM

Method :
Data preprocessing (handling missing values using backfill)
Feature selection
Model training and testing
Performance evaluation and visualization

Results : Linear Regression performed the best for this dataset, providing reliable yield predictions.
