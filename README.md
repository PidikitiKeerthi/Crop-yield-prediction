Crop Yield Prediction

Overview : This project predicts US corn yield using machine learning models based on weather, soil, and environmental data.

Models Used :
Linear Regression
Decision Tree
Random Forest
SVM
CNN, RNN, LSTM

Method :
Data preprocessing (handling missing values using backfill)
Feature selection
Model training and testing
Performance evaluation and visualization

Results : Linear Regression performed the best for this dataset, providing reliable yield predictions.

Research Paper :
This project is based on our research paper titled "Crop Yield Prediction using Deep Learning", presented at an IEEE conference.
The study explores various machine learning and deep learning models for predicting crop yield using agricultural data.
Paper Link: https://ieeexplore.ieee.org/abstract/document/10841731
